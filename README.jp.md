# Docker-Unlock
ロシア、キューバ、イラン、北朝鮮、クリミア共和国、スーダン、シリアでのDocker Hubのロックを解除します。

### 使い方
```bash
./unlock.sh
```

> [!IMPORTANT]\
> スクリプトを実行可能にすることを忘れないでください:
>```bash
>  chmod +x unlock.sh
> ```
